现有如下30个灾难地点，需要安排车辆去派送物资，现知道车库地点坐标o（35,45），货物仓库地点cage（40,50），

现车库有车56辆，要求分成8个车队（每个车队7辆车，不考虑车的差异，每辆车载重为6，平均车速40）去派送物资，而且时间紧张，每个灾难地点有时间限制。每次装卸物资需要耗时0.5。

要求：

①如果车上物资不够下一个地点使用，回到仓库装满物资再出发

②一个车队要求一起行动，不可分开。

现使用遗传算法求解如何规划使得最优？

L是车队车辆数量，S是车队行驶路程，$T_超$是总超时时间，N是超时次数。

class OneInit:
    def __init__(self):
        self.ans = None
        self.z = 0
        self.s = 0
        self.all_t = 0

    def form_ans(self):
        # random.seed(self.seds)
        self.ans = [[] for _ in range(k)]
        rand_data = random.sample(list(sort_data), c)
        for i in range(c):  # c 30
            self.ans[i % 8].append(rand_data[i])

        # for i in self.ans:

    def limited(self):
        # 初始化一个个体，1/80 得到对应zs
        zs = []
        for i in range(k):
            # 单车队初始
            s = 0  # 路程
            out_t = 0  # 总超时
            all_t = 0  # 时间沙漏
            temp_g = l * g
            start_p = cage
            out_num = 0
            # 出仓库
            all_t += tx  # 装货
            for ixx in self.ans[i]:
                if temp_g >= ixx[2]:
                    s1 = self.Edis(start_p, ixx[-2:])
                    s += s1
                    all_t += s1 / v
                    if all_t - ixx[1] > 0:
                        out_num += 1
                        out_t += max(0, all_t - ixx[1])
                    # 卸货
                    all_t += tx
                    temp_g -= ixx[2]
                    start_p = ixx[-2:]
                else:
                    s1 = self.Edis(start_p, cage)
                    s += s1
                    all_t += s1 / v
                    all_t += tx
                    start_p = cage
                    temp_g = l * g
            zs.append(l * s + out_t * alpha + out_num * beta)
            self.s += s
            self.all_t += all_t
        self.z = sum(zs)

    def run(self):
        self.form_ans()
        self.limited()

    @staticmethod
    def Edis(x1, x2):
        return pow(pow(x1[0] - x2[0], 2) + pow(x1[1] - x2[1], 2), 0.5)

开始建立第一代喽（采用随机的形式建立80个个体，每一个个体是一个问题的解）

我们取出一个个体来看一下他的样子，方便理解

上图表示一个个体，一共8条基因，对应8个车队 一条基因对应车队负责的3或4个点，一开始先这么设置。后面会发生交换和突变 基因中3/4点可能会变，哦不 一定会变的！

第一代，我们先初始化这样的80个个体，就是相当于有了80个随机解，这些解坑定大部分都是垃圾解，但是也有可能有那么一个两个好点的，那么如果让好的解变多，差的解变少呢，哎 达尔文的理论来了，“优胜劣汰“，没错 定义一种惩罚规则 让不好的个体死去，通过个体基因变化产生新个体，如此一代一代变异，淘汰……发展啊发展最终停在某个地方，当然你可以自己设计到底迭代多少次，也可以让机器控制，这里我们手动控制，迭代40次。然后在最新的一代随便拿出一个个体吊打第一代好吧，我们在最新的一代挑一个最健壮的（也就是适应度最高的）拿出来，这个解就是我们问题的解。

过程原理说完了，代码接着往下走

④变异交换和淘汰（交换概率0.8，变异概率0.1）

这里提供两种基因交换思路，一种变异思路

1、自身交换：

当然这种交换随机的 这里是一个例子，这里是交换了一个，也可以是23号和4号一起断裂，插入到另一个地方。

2、基因间交换

这两种交换都是为了产生更多的解 但是和原种群又不能差别太大，理解这种思路就行，不要问我为什么基因会这样交换！！！问就是为了产生更多相似解，不要问我生物，不要生物，不要…

这个例子举的比较极限，但是是真的会发生的。

3、变异

4、淘汰

这里因为进化策略是80个生成80个 就是160个

淘汰法选择局部淘汰：让随机两个个体博弈，适应度低的死，活下来的80个新的（里面可能有上一代旧的）整体应该比上一代要好一些。

def evolute(samples1):
    new_samples = samples1.copy()
    for sami in range(num):
        body = samples1[sami].copy()
        for _ in range(1):
            j = random.sample([i for i in range(k)], 1)[0]
            ind = random.sample([i for i in range(k)], 1)[0]
            if j == ind:
                break
            if len(body[j]) <= 2 or len(body[ind]) <= 2:
                continue
            rad = [random.random() for _ in range(2)]
            if rad[1] < pe:
                st1 = random.sample([0, -1], 1)[0]
                st2 = random.sample([0, -1], 1)[0]
                if st2 == 0:
                    body[ind] = [body[j][st1]] + body[ind]
                elif st2 == -1:
                    body[ind] = body[ind] + [body[j][st1]]
                if st1 == 0:
                    body[j] = body[j][1:]
                elif st2 == -1:
                    body[j] = body[j][:-1]
            if len(body[j]) <= 4 or len(body[ind]) <= 4:
                continue
            ind = random.sample([i for i in range(k)], 1)[0]
            try:
                sel1 = sorted(random.sample([i for i in range(len(body[j]) - 1)], 2))
                sel2 = sorted(random.sample([i for i in range(len(body[ind]) - 1)], 2))
            except ValueError:
                continue
            if sel1[1] - sel1[0] > len(body[j]) - 2 and sel2[1] - sel2[0] > len(body[ind]) - 2:
                continue
            if rad[0] < pc and len(body[j]) >= 3 and len(body[ind]) >= 3:
                e = body[j][sel1[0]:sel1[1] + 1]
                del body[j][sel1[0]:sel1[1] + 1]
                body[j] = body[j][:sel1[0]] + body[ind][sel2[0]:sel2[1] + 1] + body[j][sel1[0]:]
                body[ind] = body[ind][:sel2[0]] + e + body[ind][sel2[1] + 1:]

        vum = sum([len(qqq) for qqq in body])
        if vum != 30:
            new_samples.append(samples1[sami])
        else:
            new_samples.append(body)
    return new_samples


# 限制条件  总路程   花费总时间
def eliminate(data, zs, ss, ts):
    for i in range(num):
        e = OneInit()
        e.ans = data[num + i]
        e.limited()
        zs.append(e.z)
        ss.append(e.s)
        ts.append(e.all_t)
    return zs, ss, ts

% clear all;
% 灾难点位置 xy坐标系
ix = [2, 4, 5, 5, 7, 7, 10, 10, 12, 14, 14, 16, 20, 20, 20, 22, 22, 24, 25, 26, 26, 30, 36, 48, 54, 55, 56, 57, 60, 68];
iy = [40, 18, 35, 46, 40, 45, 40, 35, 24, 75, 80, 75, 25, 80, 85, 85, 75, 58, 85, 54, 35, 67, 46, 43, 44, 54, 38,48,53, 5];
% 时间限制和需求货物
ls = [5.59, 2.55, 3.9, 5.61, 3.25, 3.56, 2.65, 3.20, 3.4, 4.28, 2.26, 2.33, 5.59, 3.81, 2.40, 3.04, 2.71, 2.58,5.48,3.28, 5.81, 2.84, 5.73, 3.32, 2.49, 4.88, 4.56, 2.16, 2.88, 5.75];
gs = [10, 15, 5, 20, 10, 10, 10, 5, 10, 15, 20, 10, 5, 5, 10, 14, 6.5, 9.5, 12.5, 3, 6.5, 7, 1.5, 11.5, 3, 13, 17.5,4.5, 7.5, 1.5];
% 出发点
o = [35,45];
% 仓库位置
cage = [40,50];
% 出发点与仓管库位置欧氏距离
o_cage = sqrt(pow2(o(1)-cage(1))+pow2(o(2)-cage(2)));

c = 30;  % 灾难点数量
l = 7;  % 车数量
k = 8;  % 车队数量
g = 6;  % 每辆车载重
v = 40;  % 车速
tx = 0.5;  % 装卸时间

% 一些预处理工作
% disp("灾难点编号，时间限制，要求物资量，灾难点x轴，灾难点y轴");
data = [[1:c];ls;gs;ix;iy]';
sort_data = sortrows(data,2);
o_cage_t = o_cage / v;
% 提前处理前往仓库消耗时间
sort_data(2) = sort_data(2)-o_cage_t;

% 开始遗传算法
% 初始化种群
num = 80;   % 种群个体数量
pc = 0.8;      % 交换概率
pe = 0.1;     % 变异概率
alpha = 40;   % 约束函数参数
beta = 100;     % 约束函数参数
epoch = 20;    % 总种群进化次数（迭代次数=epoch*num）

% 建立初代
zs = [];  % 限制条件
samples = zeros([10,(5*k*num)]);  % 种群
ss = [];  % 总路程
ts = [];  % 花费总时间

% 为了记录最优解设置随机数种子
seed = unidrnd(9999999);
% 可以自己调
rand('seed',180279);
for i=1:num
    [ansi,zsi,ssi,tsi] = constraint([],k,c,l,g,tx,v,cage,sort_data,alpha,beta);
    zs(i) = zsi;
    ss(i) = ssi;
    ts(i) = tsi;
    samples(:,5*k*i-(5*k-1):5*k*i) = ansi;
end

% 迭代进化
for w =1:epoch
    disp(['=======>epoch : ',num2str(w*num)]);
    samples = evolute(samples,pc,pe,k,num);
    % 计算新一代适应度
    for i=1:num
        [ansi,zsi,ssi,tsi] = constraint(samples(:,k*5*(i-1)+1:k*5*i),k,c,l,g,tx,v,cage,sort_data,alpha,beta);
        zs(num+i) = zsi;
        ss(num+i) = ssi;
        ts(num+i) = tsi;
    end
    % 竞争淘汰1/2个体 160->80
    rand_index = randperm(num*2);
    del_index = zeros([1,num]);
    temp_samples = zeros([10,3200]);
    for i=1:2:num*2
        players = [rand_index(i),rand_index(i+1)];
        if zs(players(1)<=zs(players(2)))
            del_index(floor(i/2)+1) = players(2);
            temp_samples(:,floor(i/2)*5*k+1:(floor(i/2)+1)*5*k) = samples(:,(players(1)-1)*5*k+1:players(1)*5*k);
        else
            del_index(floor(i/2)+1) = players(1);
            temp_samples(:,floor(i/2)*5*k+1:(floor(i/2)+1)*5*k) = samples(:,(players(2)-1)*5*k+1:players(2)*5*k);
        end
    end
    zs(del_index) = [];
    ss(del_index) = [];
    ts(del_index) = [];
    samples = temp_samples;
    disp(['<适应度> : ',num2str(mean(zs))]);
    disp(['<总路程> : ',num2str(mean(ss))]);
    disp(['<总时间> : ',num2str(mean(ts))]);
end
% 输出最后一代 最健壮的个体
[z,best_sample_index] = min(zs);
disp("最优规划 : ");
disp(samples(:,(best_sample_index-1)*5*k+1:best_sample_index*5*k));
disp(['<适应度> : ',num2str(z)]);
disp(['<总路程> : ',num2str(ss(best_sample_index))]);
disp(['<总时间> : ',num2str(ts(best_sample_index))]);

%画图
best_body = samples(:,(best_sample_index-1)*5*k+1:best_sample_index*5*k);
marker = ['-y','-m','-c','-r','-g','-b','-w','-k'];
for i =1:k
    x = [o(1),cage(1),best_body(:,i*5-1)'];
    x(find(x==0))=[];
    y = [o(2),cage(2),best_body(:,i*5)'];
    y(find(y==0))=[];
    plot(x,y,marker(i));
    hold on;
end
plot([o(1),cage(1)],[o(2),cage(2)],'<');
hold on;
plot(sort_data(:,4),sort_data(:,5),'*');
hold on;
