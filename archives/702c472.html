推荐给刚开始使用Python的小白

发表于 2021-04-26
  • Lab Chen
首页 技巧

python 如何给自己的电脑搭建开发环境，只是介绍经验，不是介绍如何安装（安装方法自行百度）

小白食用。

最近给刚入门的人配乱七八糟的，调环境，随手写。

不知道你现在用什么写代码，看完这篇相信你会对 python 的开发环境有一个更好的理解
首先给出几个选择

Anaconda+pycharm或jupyter notebook或其他

适合自己的
安装 python 的时候一定要考虑好自己安装 python 是长期用，还是说简单的学习一下：
如果不长期使用：

①直接在 python 官网下载想要的版本即可
（ps：如果要玩神经网络，深度学习之类的建议 python3.6，其他的建议 python3.7。至于为什么不推荐最新的 3.8，等你用的模板多了，有的模板版本之间会有冲突，而你要找解决办法，网上大多都是 3.7 3.6.. 靠前的版本的解决方案，有的并不适用于 3.8）
②配置路径 path（方法网上都有，这里就不多说）

如果长期使用：

①Anaconda 官网下载后安装（具体安装的时候各种选项 bd 都有，Anaconda 的使用好处，他可以设置多个虚拟环境，环境出问题可以直接删除，而且安装 Anaconda 后不用去 python 官网下载 python，配置虚拟环境会直接安装）
②安装好了会有一个命令台图标（name：“Anaconda Prompt (Anaconda3)”，打开直接使用下面命令）

conda create -n name python=3.7
# name是虚拟环境名字，可以随便换
# 3.7是版本，也是可以换的

虚拟环境配好后用下面命令切换到新建的虚拟环境就可以使用 pip 或 conda 安装各种喜欢的库了

activate name
# name是环境名字

如果是 conda 安装，下载速度大部分情况是没问题
如果是 pip 安装，速度会有些慢，建议后面加一个镜像源：

例子：
pip install numpy -i https://pypi.tuna.tsinghua.edu.cn/simple
PS：使用清华镜像源安装numpy库

③安装 pycharm/jupyter notebook / 其他
这些东西都属于集成开发环境，都有不同的偏向：
pycharm（官网下载即可）：
优点：适合写一个项目工程
缺点：虽然有内置的 console 可以有一个交互体检，但是不能单独运行一个代码块
jupyter notebook（这是一个 python 库，用 pip 就可以安装）：
优点：可以单独运行代码块，有点类似 MATLAB，每次运行参数会保留在一个工作空间，可以单独查看
缺点：文件类型是 ipynb，只能用 jupyter notebook 打开，其他打开需要转化成 py 文件
其他：
ptpython（pip 可以安装）：有交互操作（我是把这个当小工具用的，比如突然想研究某个函数又懒得建个项目）
其他还有很多，用的多了会有自己单独的见解。
