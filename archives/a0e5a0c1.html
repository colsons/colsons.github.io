# dos 使用 txt 文件批量安装库


pip.txt 内容就是这样

在桌面新建一个 txt 文件后缀改成 bat，然后记事本打开编辑内容：

cd /dZ:

for /f %%i in (pip.txt) do start "" cmd /k "activate django207&pip install %%i -i https://pypi.tuna.tsinghua.edu.cn/simple"

::taskkill /f /im cmd.exe /t
::将上面一行命令关闭所有命令窗口
